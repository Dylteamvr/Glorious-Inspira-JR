# GLORIOUS INSPIRA JR

GLORIOUS INSPIRA JR is a VRChat AI Bot built to explore virtual spaces, react to players, and learn from interaction in real time.

Created and developed by **Dylteam**.

It combines:
- OCR vision
- autonomous movement
- chat learning
- voice input
- emotional behavior modes
- player tracking
- unstuck recovery

## What it does

GLORIOUS INSPIRA JR can:
- move around on its own
- detect players and chat text
- learn words and phrases
- respond with simple generated messages
- switch between different behavior modes
- recover when it gets stuck
---------------------------------------------------------------
## Features

### Vision
Uses OCR to try to detect player names and chat text on screen.

### Movement
Includes realistic movement patterns like:
- walking
- strafing
- sprinting
- jumping
- circling
- exploration

### Learning
Stores phrases and words so the AI can build a simple memory over time.

### Modes
Includes optional modes such as:
- Emotional Mode
- Chaos Mode
- Follow Mode
- Prank Mode
- Stand Still Mode
